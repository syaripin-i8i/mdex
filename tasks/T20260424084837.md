## Done When

1. `context_scoring` セクションを config から読めるようになり、未指定時は現行定数を使う
2. runtime config と scan config の優先順が定義され、どの設定を採用したか `score_breakdown` から辿れる
3. `tests/fixtures/quality_repo` を使った ranking 回帰テストが追加される
4. `docs/context_scoring.md` に tuning の入口と `config_source` の意味が追記される

- デフォルト順位を理由なく全面的に入れ替えること
- 学習ベースの自動調整を導入すること

## Result

## コマンド仕様

### `scan`

```
python runtime/cli.py scan --root <dir> [--output <file>] [--config <json>]
```

- `--root`: 走査するディレクトリ（必須）
- `--output`: 出力JSONファイルパス（デフォルト: `mdex_index.json`）
- `--config`: scan_config.json のパス（デフォルト: `control/scan_config.json`）
- 実行後: ノード数・エッジ数を stderr に出力（例: `[mdex] 12 nodes, 8 edges → mdex_index.json`）

### `query`

```
python runtime/cli.py query --index <file> --node <id> [--depth <n>]
```

- `--index`: インデックスJSONファイルパス（デフォルト: `mdex_index.json`）
- `--node`: ノードID（ファイルの相対パス）
- `--depth`: 辿るホップ数（デフォルト: 1）
- 出力: 対象ノードの JSON + depth 以内の隣接ノード一覧（JSON）

### `list`

```
python runtime/cli.py list --index <file> [--type <type>] [--project <name>] [--status <status>]
```

- フィルタ付きでノード一覧を出力（id + title + type + status の1行サマリー）

# Task: [mdex/0.1.1] pyproject の Changelog URL を default branch に揃える

**ID**: T20260424084804
**Project**: mdex
**Priority**: high
**Status**: pending
**Created**: 2026-04-24T08:48:04+09:00

## Description

実装先: `pyproject.toml`

## Background

`pyproject.toml` の `[project.urls]` にある `Changelog` が `blob/main/CHANGELOG.md` を指している一方、現行リポジトリの default branch は `master`。PyPI metadata から辿るリンクが 404 になる可能性がある。

## Done When

1. `Changelog` URL が実在する default branch に揃っている
2. `Repository` / `Issues` / `Security` を含む `project.urls` 全体を見直し、branch 名だけが孤立していない
3. `python -m build` 後の metadata で URL が期待どおりであることを確認できる

## Non-goals

- default branch 自体を rename すること
- README の badge や通常リンクを不要に張り替えること

## Result

- 実施内容: `pyproject.toml` の `[project.urls]` `Changelog` を `blob/master/CHANGELOG.md` へ修正し、`Repository` / `Issues` / `Security` と同じ default branch 前提へ整合。
- 確認方法: `python -m build` 後に wheel metadata (`*.dist-info/METADATA`) の `Project-URL` を確認し、`Changelog` が `.../blob/master/CHANGELOG.md` になっていることを確認。
- 備考: default branch rename は行わず metadata 側のみ修正。

### 2. `mdex finish --dry-run` の追加

新コマンド:

```bash
mdex finish --task "<task>" [--changed-files-from-git] [--dry-run]
```

最低限返すもの:

- `task`
- `changed_files`
- `enrich_candidates`
- `recommended_next_actions`
- `scan` 状態

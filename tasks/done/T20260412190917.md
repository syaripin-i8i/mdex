## 変更内容

### 1. 共通 DB 解決層の追加

以下の責務を持つ共通層を追加する。

- CLI 引数 `--db`
- 環境変数 `MDEX_DB`
- `.mdex/config.json`
- repo root 既定候補

read/write モード差分もこの層で扱うこと。

### 2. CLI 全コマンドへの適用

対象:

- `list`
- `open`
- `query`
- `find`
- `orphans`
- `stale`
- `related`
- `first`
- `context`
- `enrich`
- `scan`

`scan` は write モードとして扱い、`--db` 省略時でも既定 DB へ書けるようにする。

### 3. `.mdex/config.json` の取り扱い

最小限以下のキーを読む。

```json
{
  "db": ".mdex/mdex_index.db",
  "scan_root": ".",
  "scan_config": "control/scan_config.json"
}
```

相対パスは repo root 基準で解決すること。

### 4. エラー JSON の改善

解決失敗時は `resolution_attempts` と `hint` を返す。

## 目的

`runtime/parser.py` を新規作成する。
単一の Markdown ファイルを受け取り、構造化データを返す純粋なパーサー。

## 実装対象

`parse_file(path: str) -> dict` を実装する。

抽出する情報:
- `title`: 最初の `# 見出し`、なければファイル名（拡張子なし）
- `frontmatter`: `---` ブロックの YAML を dict で返す（PyYAML 使用）
- `wikilinks`: `[[...]]` パターンを全抽出。`[[label|alias]]` の場合は `label` を使う
- `md_links`: `[text](path.md)` 形式の内部リンク（`.md` で終わるもののみ）
- `summary`: 最初の非見出し・非frontmatterの段落（最大3文、200文字以内）
- `headings`: h2以降の見出しテキスト一覧
- `tags`: frontmatter の `tags` フィールド（list）
- `updated`: frontmatter の `updated` フィールド、なければファイルの mtime を ISO 文字列で

## 完了条件

  - frontmatter に type/project/status が含まれる
  - wikilinks に "AGENT" が含まれる

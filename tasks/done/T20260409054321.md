### 2. CLI を要約更新コマンドに変更
- mdex enrich <node-id> --db <db> --summary "<text>"

- --path <絶対パス> オプションは維持（node-id 逆引き）
- summary 未指定時は JSON エラーを返す（スキップではなくエラー）
- --force は「既存 agent summary を上書き」に限定

- summary_updated TEXT（ISO 8601、enrich 実行時刻）
- scan 時は summary_source = "seed" で書き込む
- enrich 時は summary_source = "agent" で書き込む

### 4. store.py を更新
- update_node_summary(db_path, node_id, summary, source="agent") を追加
- get_node() が summary_source / summary_updated を返すように変更
- indexer.py の write_sqlite() が summary_source = "seed" で書き込むように変更

### 5. AGENT.md / README / docs/convention.md の enrich 説明を修正

- convention.md の「エージェントへの指示」セクションの enrich 呼び出し例を更新

### 6. テスト追加
- enrich --summary で summary 更新・summary_source が "agent" になる
- enrich --summary-file で summary 更新
- summary 未指定時に JSON エラー

## 設計方針
- node-id の本文を reader.read_node_text() で取得
- Claude API（ANTHROPIC_API_KEY 環境変数）に本文を渡し、2〜3文のサマリを生成
- 生成したサマリを DB の nodes.summary に上書き保存
- API キーがない場合はスキップ（エラーにしない）
- すでに summary が充実している場合は --force フラグがないと上書きしない

- store.py に update_node_summary(db_path, node_id, summary) を追加
- cli.py に _cmd_enrich と subparser enrich を追加
  - 引数: node（位置引数）, --db, --force
  - 成功時: [mdex] enriched: <node-id> を stderr に出力
- AGENT.md の「Verification」セクションに enrich の使い方を追記
  - 「ファイルを読んだ後: mdex enrich <node-id> --db <db>」
